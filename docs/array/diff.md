# diff

Returns the difference between two arrays.

## Usage

Given two arrays, returns an array of all items that exist in the first array but do not exist in the second array.

```ts
const a = [1, 2, 3, 4, 5]
const b = [2, 4, 6, 8, 10]

diff(a, b)
// => [1, 3, 5]
```
